# Unbundl Frontend Assessment

## Setup Instructions

1. Install dependencies:
   npm install

2. Run project:
   npm run dev

## API Used

Fake Store API:
https://fakestoreapi.com/products

## Approach

I built a responsive product listing section based on the Figma design using React functional components. Data is fetched using useEffect and managed with useState, including loading and error states. The layout is fully responsive and includes a hover interaction for product cards.
